## Testing

Now you're ready for testing!

First, compile the Typescript file with:

```bash
$ pnpm build
```
To run all tests:

```bash
$ pnpm test
```
Or, to tun a specific test:

```bash
$ pnpm nightwatch <test-file-or-test-folder-path>
```
